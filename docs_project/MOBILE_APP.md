# Приложение на телефон (Samsung + iPhone)

Badger CFO — это **PWA**: устанавливается как обычное приложение (иконка на
главном экране, полный экран), один код работает и на Android, и на iOS.
App Store и Google Play не нужны.

Чтобы приложение работало с телефона всегда и откуда угодно, сервер должен
жить в облаке. Ниже — самый простой путь для новичка, через **Render.com**.

## Часть 1. Поднять сервер в облаке (10 минут)

1. Залей этот проект на свой GitHub (если ещё не там).
2. Сгенерируй два секрета на своём компьютере:
   - Ключ шифрования:
     ```
     python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"
     ```
   - Код доступа в приложение — придумай сам, например 6 цифр. **Без него
     любой, кто узнает ссылку, увидит твои финансы.**
3. Зайди на https://render.com, зарегистрируйся (можно через GitHub).
4. Нажми **New + → Blueprint**, выбери свой репозиторий. Render прочитает
   файл `render.yaml` и предложит создать сервис `badger-cfo`.
5. Перед запуском задай переменные окружения (Environment):
   - `FERNET_KEY` — вставь сгенерированный ключ
   - `APP_ACCESS_CODE` — твой код доступа
   - `TELEGRAM_BOT_TOKEN` — токен бота (если нужен и бот; для приложения не обязателен)
6. Нажми **Apply / Create**. Через пару минут Render даст ссылку вида
   `https://badger-cfo-xxxx.onrender.com`.
7. Открой эту ссылку в браузере — увидишь экран входа, введи код доступа.

> Бесплатный план Render «засыпает» при простое (первый заход потом
> медленный). План Starter (~$7/мес) держит сервис всегда живым.

Альтернативы Render с тем же Dockerfile: Railway, Fly.io, любой VPS
(DigitalOcean/Hetzner ~$5/мес) — принцип тот же.

## Часть 2. Установить как приложение

### iPhone (Safari)
1. Открой ссылку сервиса в **Safari** (именно Safari, не Chrome).
2. Нажми кнопку **Поделиться** (квадрат со стрелкой внизу).
3. Выбери **На экран «Домой»** → **Добавить**.
4. На главном экране появится иконка Badger CFO — открывается на весь экран.

### Samsung / Android (Chrome)
1. Открой ссылку в **Chrome**.
2. Меню (три точки справа вверху) → **Добавить на главный экран** /
   **Установить приложение**.
3. Подтверди — иконка появится в меню приложений как обычное приложение.

Готово. Дальше приложение запускается с иконки, помнит вход (код вводится
один раз), и работает как нативное.

## Безопасность

- Всё read-only: приложение не переводит деньги, не торгует, не выводит крипту.
- Доступ закрыт кодом (`APP_ACCESS_CODE`). Меняй его в любой момент в
  настройках Render — старые сессии слетят.
- Секреты (Plaid-токены, ключи бирж) хранятся зашифрованными.
- Используй сервис только по HTTPS-ссылке (Render даёт её автоматически).
