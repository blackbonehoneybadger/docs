## The NGY token — the grown-up approach

In version 1.0, Nagayna included an NGY token and an ICO **first**. In 2.0 the logic is more mature: **the token is not the first step.**

First comes:

- the product;
- users;
- statistics;
- usefulness;
- security;
- reputation;
- audits;
- legal consultation.

**Only then** can a token be considered.

NGY *may* become a:

- utility token
- governance token
- access token
- rewards token
- token for a strategy marketplace
- token for DAO voting
- token to pay for premium features

### What must never be promised

- guaranteed price growth;
- guaranteed profit;
- risk-free passive income;
- "buy now — get rich later".

### The correct framing

> "NGY may be used in the future as a utility instrument inside the Nagayna ecosystem, **if** the product reaches a sufficient level of users, security, and legal readiness."
