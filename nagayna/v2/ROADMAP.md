# Roadmap

Development is phased so that capability and trust grow together. The token comes **last**, after the product proves its value.

---

## Phase 1 — Foundation
**Goal:** a safe AI trading journal + dashboard.

- dashboard
- portfolio tracker
- manual trade journal
- paper trading
- basic indicators
- Telegram alerts
- daily report
- trade memory

## Phase 2 — TradingView + AI Analysis
**Goal:** connect TradingView and AI analysis.

- TradingView webhooks
- AI analysis of signals
- multi-timeframe analysis
- 20–30 indicators
- risk score
- trade recommendation
- decision explanation

## Phase 3 — Exchange Read-Only
**Goal:** connect real exchanges with no trading rights.

- Bybit read-only
- Coinbase read-only
- Kraken read-only
- trade history
- balance
- PnL
- real-portfolio analysis

## Phase 4 — Demo & Paper Learning
**Goal:** let Nagayna train.

- strategy simulator
- paper mode
- demo account
- weekly learning review
- strategy versions
- mistake analysis

## Phase 5 — Confirmation Trading
**Goal:** trades only after user confirmation.

- order preparation
- user confirms each trade
- stop/take automatically proposed
- Risk Engine blocks dangerous trades

## Phase 6 — Advanced Intelligence
**Goal:** make Nagayna smarter.

- 100 indicators
- news scanner
- on-chain analytics
- whale alerts
- DeFi scanner
- strategy marketplace

## Phase 7 — NGY Ecosystem
**Goal:** token **only after** the product.

- utility token
- premium access
- DAO
- marketplace
- governance
- audits
- legal review

---

## The first MVP

The first MVP must not be huge. It must be **real**.

Minimum first product:

- web application
- asset list
- manual trade entry
- paper trading
- 10–20 indicators
- AI explanation
- daily report
- Telegram alerts
- memory base
- risk calculator
- TradingView webhook

> **Main MVP goal:** teach Nagayna to look at the market, explain ideas, save decisions, and learn from results — **without risking real money**.

This maps directly to **Phase 1 + the early parts of Phase 2**: prove the analysis-explain-remember-learn loop in a fully safe environment before any real funds are involved.
